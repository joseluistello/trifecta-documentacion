# Como funciona Trifecta

Trifecta es una herramienta de análisis de datos que permite a los usuarios visualizar y manipular datos de múltiples fuentes. Ofrece una interfaz gráfica de usuario (GUI) para visualizar los datos en una cuadrícula, así como una interfaz de línea de comandos (CLI) para realizar operaciones de línea de comandos en los datos. Trifecta también proporciona una API REST para realizar operaciones de línea de comandos en los datos.
